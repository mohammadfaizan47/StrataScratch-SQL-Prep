# StrataScratch-SQL-Prep
This course includes all Questions of SQL, which can be asked to a Data Engineer Fresher/Intern..

This course consists of Four Phases: \
• Phase 1 → Foundations \
                        ├── DataRetrieval Basics \
                        ├── Filtering Data \
                        ├── Distinct Counts \
                        └── Sorting Results \
• Phase 2 → Core Skills \
                        ├── Joins & Set Operations \
                        ├── Aggregate Functions \
                        ├── Aggregations \
                        ├── Grouped Aggregates \
                        └── Null Handling \
• Phase 3 → Window Functions \
• Phase 4 → Date & Time + String
